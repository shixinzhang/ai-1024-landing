# AI_1024 专栏介绍页

## 🚀 一键部署到 Vercel

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/shixinzhang/ai-1024-landing)

## 📁 文件说明

- `index.html` - 完整版介绍页面
- `wechat.html` - 微信友好引导页

## 🔗 访问地址

部署后可通过以下地址访问：
- `https://你的域名/` - 完整版
- `https://你的域名/wechat.html` - 微信引导页

## ⚡ 快速开始

1. 点击上方 **Deploy with Vercel** 按钮
2. 使用 GitHub 账号登录 Vercel
3. 选择项目名称，点击 Deploy
4. 等待部署完成，即可获得国内访问友好的地址

## 📱 微信访问建议

微信内访问时，推荐使用 `/wechat.html` 路径，该页面针对微信内置浏览器优化。
